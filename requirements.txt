#
# WINDOWS:
# --------
# 1. Tesseract OCR:
#    - Download from: https://github.com/UB-Mannheim/tesseract/wiki
#    - Add installation directory to PATH (e.g., C:\Program Files\Tesseract-OCR)
#
# 2. VLC Media Player:
#    - Download from: https://www.videolan.org/vlc/
#    - Required for video audio playback
#
# 3. FFmpeg (for MP3 export):
#    - Download from: https://ffmpeg.org/download.html
#    - Add bin folder to PATH
#    - Or use: winget install ffmpeg
#
# MACOS:
# ------
# brew install tesseract ffmpeg
# Download VLC from https://www.videolan.org/vlc/
#
# LINUX (Ubuntu/Debian):
# ----------------------
# sudo apt update
# sudo apt install tesseract-ocr vlc ffmpeg python3-tk
#
# LINUX (Fedora):
# ---------------
# sudo dnf install tesseract vlc ffmpeg python3-tkinter
#
# =============================================================================
# PIPER TTS VOICES
# =============================================================================
#
# Download voice models using the included script:
#   python download_voices.py --us-only --quality medium
#
# Or download manually from:
#   https://huggingface.co/rhasspy/piper-voices
#
# Voice files should be placed in:
#   Windows: %LOCALAPPDATA%\piper-voices
#   macOS/Linux: ~/piper-voices
#
# =============================================================================
# MOONDREAM AI MODEL
# =============================================================================
#
# The Moondream vision model (~1.7GB) will be automatically downloaded
# on first run and cached in:
#   Windows: %USERPROFILE%\.cache\huggingface
#   macOS/Linux: ~/.cache/huggingface
#
# First launch may take several minutes to download the model.
# Subsequent launches will load from cache (30-60 seconds).
#
# GPU SUPPORT:
# ------------
# For NVIDIA GPU acceleration, install PyTorch with CUDA:
#   pip install torch torchvision --index-url https://download.pytorch.org/whl/cu124
#
# For Apple Silicon (M1/M2/M3), PyTorch MPS is used automatically.
#
# CPU-only systems will work but inference will be slower.
